Fathym is an innovation framework for the Industrial Internet of Things, helping customers bring their data into a cloud environment and create real-time visualizations, alerts and apps, as well as optimize systems through historic analytics.

Real-Time Data Monitoring


Fathym is a flexible framework for rapidly building data dashboards, custom to your data needs. Data brokers and broker subscriptions help you connect your device data or data feeds.

Our catalog of data visualizations are created from standard languages (HTML, Javascript, JQuery) and open source libraries, such as D3.js.

The Fathym Rules Engine allows users to set alerts based on any combination of incoming data.

Data Ingest

Fathym connects to data via gateways, APIs and data brokers.

After the broker is configured, selecting a data stream for your dashboard is easy. Multiple data streams can be applied to a single dashboard.


Each data visualization allows you to select your data source from the dashboard's data streams.


Real-Time Monitoring and Data Visualizations

Users can easily add gauges, dials, charts and maps to their real-time dashboards. Each visualization can be set to display any incoming data element. Dashboards can be customized to each user or role in an organization.

Alerts & Notifications

Alerts can be set on any combination of incoming data. When data thresholds are reached, notifications can be sent via email or SMS.


Dashboards come with a management view of all alerts, who the alerts are assigned to, when they last ran, and when the alert was last triggered.


Integrating Diverse Datasets for Visualizations and Decisions

Fathym simplifies embedding external apps and dashboards through APIs and embed parts. This allows clients to bring externally crafted dashboards and apps into Fathym, along with internal datasets, under a single pane of glass.
